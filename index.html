There is a solution of task: Some guy made an app for a robot. Possible commands: S - move forward, L - turn left, R - turn right. Does a robot get back to a start position? Please, write -1 if it’s so, otherwise write “yes”

data = ‘SLSRSRSRSS’
point = [0, 0]
directions = [0, 1, 2, 3]
cur_direction = 0

dx = [0, -1, 0, 1]
dy = [1, 0, -1, 0]

for i in range(len(data)):
print(‘current position -‘, point)
if data[i] == ‘S’:
point[0] += dx[cur_direction % 4]
point[1] += dy[cur_direction % 4]
print(‘move to ‘, point)
elif data[i] == 'R':
cur_direction += 3
cur_direction %= 4
print(cur_direction)
elif data[i] == 'L':
cur_direction += 1
cur_direction %= 4
if point == [0,0]:
print('Вернулся')
else:
print(-1)
